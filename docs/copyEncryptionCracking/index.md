# copy encryption cracking 破解复制加密

破解公司加密系统禁止从 VS Code 复制。让 VS Code 在受控环境下也能稳定把内容复制到外部应用。通过 PowerShell 多格式写入系统剪贴板，尽量绕过企业加密/DLP 对剪贴板的限制。

![效果图](/assets/copy1.png)

## 使用方法

- 方案一：编辑器中选择文本，按快捷键  
  选中文本 → 按 `ctrl+q`（或右键菜单“破解复制”）

- 方案二：资源管理器中复制名称  
  在左侧资源管理器对文件/文件夹右键 → “破解复制”
  ![效果图](/assets/copy2.png)

- 方案三：输入过桥（适用于扩展市场、设置页等无法直接获取文本的区域）  
  任意页面/资源管理器右键 → 执行“破解复制（输入过桥）”  
  将要复制的文本粘到面板输入框 → 回车（或点按钮）
  ![效果图](/assets/copy3.png)

## 平台

- 仅 Windows（依赖 PowerShell `Set-Clipboard`）
